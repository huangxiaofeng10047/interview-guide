项目介绍

InterviewGuide 是一个集成了简历分析、模拟面试和知识库管理的智能面试辅助平台。系统利用大语言模型（LLM）和向量数据库技术，为求职者和 HR 提供智能化的简历评估和面试练习服务。

系统架构

系统采用前后端分离架构，后端基于 Spring Boot 4.0 开发，前端使用 React 18.3 + TypeScript 构建。简历分析、知识库向量化和面试报告生成采用 Redis Stream 异步处理，确保系统性能和可靠性。 

上传请求 → 保存文件 → 发送消息到 Stream → 立即返回
                              ↓
                      Consumer 消费消息
                              ↓
                    执行分析/向量化任务
                              ↓
                      更新数据库状态
                              ↓
                   前端轮询获取最新状态

核心功能

简历分析模块

  • 支持 PDF、DOCX、DOC、TXT 等多种简历格式
  • 基于 Redis Stream 实现异步简历分析
  • 内置分析失败自动重试机制与重复检测
  • 支持将 AI 分析结果导出为 PDF 报告

模拟面试模块

  • 基于简历内容智能生成针对性面试题目
  • 支持配置多轮智能追问，构建真实场景
  • 采用分批评估策略，规避 Token 溢出风险
  • 提供多维度改进建议与统计信息

知识库管理模块

  • 支持多种格式文档的自动上传与向量化
  • 集成向量数据库，实现 RAG 检索增强
  • 基于 SSE 技术实现打字机式流式响应
  • 支持基于知识库内容的智能问答

快速开始

环境要求

  • JDK 21+
  • Node.js 18+
  • PostgreSQL 14+
  • Redis 6+

启动步骤

1. 克隆项目

git clone https://github.com/Snailclimb/interview-guide.git
cd interview-guide

2. 配置数据库

-- 创建数据库
CREATE DATABASE interview_guide;

-- 连接数据库并启用 pgvector 扩展
CREATE EXTENSION vector;

3. 启动后端服务

./gradlew bootRun

后端服务启动于 http://localhost:8080

4. 启动前端服务

cd frontend
pnpm install
pnpm dev

前端服务启动于 http://localhost:5173

使用场景

求职者

上传简历获取分析建议，了解自己的优势和不足，进行模拟面试练习，提高面试技巧和自信心。

HR/招聘人员

批量分析简历，快速评估候选人能力，筛选合适的应聘者，提高招聘效率和准确性。

培训机构

提供面试培训服务，管理知识库资源，为学员提供个性化的面试练习和指导。

常见问题

Q: 数据库表创建失败/数据丢失

这大概率是 JPA 的 ddl-auto 配置不对的原因。对于新数据库，首次启动用 create，表创建成功后，改回 update。

Q: 知识库向量化失败

当 initialize-schema: false 时，Spring AI 不会自动创建 vector_store 表。建议开发环境设置为 true，方便快速启动。

Q: 简历分析失败

检查一下阿里云 DashScope API KEY 是否配置正确（申请地址：https://bailian.console.aliyun.com/）。

Q: 简历分析一直显示"分析中"？

检查 Redis 连接和 Stream Consumer 是否正常运行。查看后端日志确认是否有错误。
